 Django Weather Forecast App

A web-based weather forecasting application built with Django that provides real-time weather updates using external weather APIs. The system allows users to search for any location and view current weather conditions, temperature, humidity, and forecasts.

 Features

- Real-time weather data fetching using API integration
- Search weather by city or location
- Display current temperature, humidity, and weather conditions
- Multi-day weather forecast
- Clean and responsive user interface
- Error handling for invalid locations
- Simple and fast Django backend

 Tech Stack

- Backend: Django (Python)
- Frontend: HTML, CSS, Bootstrap (or custom CSS)
- API: OpenWeatherMap API (or your chosen weather API)
- Database: SQLite (default Django DB)
